Installing new functionality

It is easy to download and install additional features to Bioclipse. Open the menu Install > New Feature... and Bioclipse will contact the main Update Sites, and let you choose additional features to install.

Install your own feature

It is easy for anyone to produce their own p2 repository for Bioclipse, and you can easily add more sites to Bioclipse in the Bioclipse Preferences under the section Install/Update > Available Software Sites. To install your own feature into Bioclipse you have to add your own p2 repository and enable the advanced option in preferences. Then you can use the Install > Software from update site.

On startup, Bioclipse automatically searches for software updates if you have an Internet connection. This can be turned off in the dialog that pops up.
