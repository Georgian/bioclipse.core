Using Preferences

You administrate your account(s) in the user view of Bioclipse preferences. Choose Window > Preferences and then Users in the list to the left of the preferences dialog to open it.
The user view has a list of all your accounts and beneath it there is some buttons:
Create...Create a new user account.
Edit...Edit the user account selected in the list above.
DeleteDelete the selected user account in the list above.
Always logout without promptIf this is checked in then you don't get any warning when logging out. See also Logging in and out.
Restore DefaultsRestore the settings to default.
ApplyApply the settings. I.e. it does the same as the OK-button but with out closing the preferences window.

Create a new user account

 To create a new Bioclipse user-account click on the button labeled Create... and a dialog looking almost as the one in the section Creating a new user account appears. In the top field you fill in your (Bioclipse) username, in the middle your password and then repeat it in the filed at the bottom. Then click the OK-button.

Edit the user account
Before the edit dialog appears you will be prompt for the chosen user accounts password. Fill it in and click ok. Now the Edit-dialog (see above) will appear. To the left is a list with the different third-part accounts associated with this user account. To the right is a view with the properties of the account selected in the account list. The four buttons above and below the account list are:
Change Bioclipse Account PasswordOpens a dialog where you can change the password to this user account.
Add account...Create a new third-part account.
Edit account...Edit the properties of the selected third-part account.
Delete accountDelete the selected third-patrt account.

The account properties view

 This view contain all information about the third-part account. The uppermost is what type of account it is, e.g. OpenTox. Beneath is a spread sheet with the different properties, their value and whether it's required or not. You may change the value of the different properties by changing it in the spread sheet or by using the Edit...-button to the left. One more thing worth to notice is the checkbox called "Show password" to the right and beneath the spread sheet. Selecting it and all the hidden values will be in plain language.

Add a third-part account

 When clicking on the Add account-button a new window that look a lot like and with the same function as the one described in Add a third-part account appears. If the button is not enable, it's probably because you do not have any feature that need a web-service installed.
The first thing to do is to choose the account type, i.e. which feature you want to create a third-part account to, this is done in the dropdown-menu at the top. Then fill in the information that Bioclipse need to be able to log in to the third-part account. You may also change the name of the third-part account. Now click the Finich-button, then the new third-part account is added to your Bioclipse user-account and you return to the edit-dialog.
If you are logged in to a user account you may have to log out and then log in again to let Bioclipse log in to the new third-part account, see Logging in and out.

Edit a third part account

 Clicking on this button and a simple dialog where the value of the properties of the third-part account can be edit appears. Just change the value of the property or properties, e.g. the password, and then click ok. If you are logged in to the account when changing the properties, you may also have to log out and then log in again to the user account for the changes to take effect, see Logging in and out.
