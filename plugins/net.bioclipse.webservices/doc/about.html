About Web Services Plug-In in Bioclipse

This Plug-In is a framework to add Web Service functionality to Bioclipse. There is currently no special view or perspective for this Plug-In. The implementation of each Web Service depends on its functionality and the need. Currently there is only one Web Service supported (WSDbfetch) as it makes much sense in example to get pdb or RefSeq entries directly from the web.
